# Nano Course


## Setup

Checkout
```
git clone --recurse-submodules https://github.com/jonasschultemattler/nano.git
```

Compile
```
mkdir build
cd build
cmake -DCMAKE_BUILD_TYPE=Release .. -D CMAKE_CXX_COMPILER=g++-14
make
```

Test
```
./source/hello
```
